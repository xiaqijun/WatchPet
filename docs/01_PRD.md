# WatchPet PRD v1.0

## 产品定位

WatchPet 是 Apple Watch 上的随身电子宠物应用，将真实宠物照片转化为可互动的手表宠物，让用户在手腕上获得陪伴、养成和轻游戏体验。

## 产品目标

- 让用户把自己的猫/狗/其他宠物变成手表里的虚拟宠物。
- 提供低负担、高频次、几秒即可完成的互动。
- 用步数等健康数据增强养成反馈。
- 后续通过 AI 生成、宠物包、装扮、道具形成内容生态。

## MVP 用户故事

1. 作为用户，我打开 Apple Watch App，可以看到宠物在屏幕上待机动起来。
2. 作为用户，我点一下宠物，它会做被摸摸的动作，心情变好。
3. 作为用户，我可以喂食，让宠物不再饥饿。
4. 作为用户，我可以让宠物睡觉，恢复精力。
5. 作为用户，我走路后再次打开 App，宠物获得经验。
6. 作为用户，我长时间不打开 App，宠物会变饿或心情变低。

## 信息架构

### Apple Watch MVP

- 主屏：宠物动画、等级、状态条。
- 操作区：喂食、睡觉。
- 互动：点按宠物。

### iPhone 后续

- 宠物详情。
- 生成宠物。
- 导入宠物包。
- 同步到手表。
- 设置。

## 体验原则

- 单次互动不超过 3 秒。
- Apple Watch 屏幕上文字极少，动画优先。
- 失败/饥饿/生病也要可爱，避免过度负担。
- 健康数据必须可选，不授权也能玩。
- 动画资源必须清晰、低体积、低耗电。

## 非目标

- 不是完整游戏。
- 不是 3D 宠物。
- 不是桌面宠物。
- 不是 Codex 状态同步器。
- MVP 不做社交平台。

## 成功指标

MVP 内部测试指标：

- 首次启动 5 秒内看到宠物。
- 互动按钮 100% 有状态反馈。
- 重新打开 App 状态能保存。
- 无 HealthKit 权限时 App 可正常使用。
- 单轮动画播放不卡顿。
